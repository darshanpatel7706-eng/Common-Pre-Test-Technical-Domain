# Section D: Artificial Intelligence & Machine Learning

## Q1. Difference Between AI, ML, and Deep Learning
Using a transportation analogy:
- *Artificial Intelligence (AI)* is like vehicles that can move on their own.
- *Machine Learning (ML)* is like learning driving rules from experience.
- *Deep Learning (DL)* is like self-driving cars that learn complex patterns automatically.

---

## Q2. Spam Email Detection System
This is a *classification* problem.

Steps involved:
1. Collect labeled email data (spam or not spam)
2. Clean and preprocess the data
3. Extract features from text
4. Train a classification model
5. Test and deploy the model

---

## Q3. Training Accuracy vs Testing Accuracy
This problem is called *Overfitting*.

Two ways to address it:
- Use regularization techniques
- Increase training data or apply cross-validation

---

## Q4. Teaching a Child vs Training an ML Model
Similarities:
- Learning from examples
- Recognizing patterns
- Making mistakes and improving

Differences:
- ML models need large datasets
- ML models do not have emotions or intuition

## Q5. Recommendation System for Online Store
Data required:
- User purchase history
- Browsing behavior
- Product ratings

The system can use collaborative filtering to suggest relevant products.
