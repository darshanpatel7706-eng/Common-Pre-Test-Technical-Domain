# Section E: Cyber Security & Infrastructure

## Q1. Identifying a Phishing Email
Red flags to identify a phishing email:
- Unknown or suspicious sender
- Urgent or threatening language
- Fake or shortened links

---

## Q2. Risk of Using Same Password
Using the same password on multiple websites is risky because if one site is compromised,
all accounts become vulnerable.

Safer practice:
- Use unique passwords
- Use a password manager

---

## Q3. Traditional Server Room vs Cloud Computing
*Advantages of Cloud Computing:*
- Easy scalability
- Cost efficiency

*Potential Concern:*
- Data privacy and security

---

## Q4. CI/CD in DevOps
CI/CD stands for Continuous Integration and Continuous Deployment.

It automates:
- Code testing
- Application deployment

This allows faster and more reliable software updates.

---

## Q5. Website Security Measures
To protect sensitive customer data, I would implement:
- HTTPS encryption
- Firewalls
- Secure authentication mechanisms
